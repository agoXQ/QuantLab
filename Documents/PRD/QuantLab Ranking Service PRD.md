# 模块定位

Ranking Service 是 QuantLab 的策略评价与发现中心。

负责：

- 策略排名计算

- 策略评分体系

- 榜单生成

- 榜单展示

- 榜单历史管理

- 榜单反作弊

不负责：

- 回测执行

- 用户管理

- 社区互动

# 策略可信度体系（Strategy Trust Score）

## 设计目标

防止高收益低可信策略污染排行榜。

---

## 可信度计算维度

回测年限

交易次数

收益稳定性

回撤控制

长期上榜记录

---

## 可信度公式

```Plain Text
Trust Score =

回测年限得分（0-20）
+
交易次数得分（0-20）
+
收益稳定性得分（0-20）
+
回撤控制得分（0-20）
+
长期上榜得分（0-20）
```

满分 100 分。

---

## 展示方式

榜单中同时展示：

```Plain Text
收益率：300%

可信度：92分
```

---

## 准入规则补充

可信度低于 60 分的策略：

不进入综合榜

仅进入收益榜（标记风险提示）
